# 落霞合成器

## 简介

落霞是一款基于 wlroots 的 Wayland 合成器与窗口管理器，支持用 VNC 协议将画面传送到远程桌面。可作为虚拟实验平台的组件。

该项目为作者毕业设计项目的一部分。
